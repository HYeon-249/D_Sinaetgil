iOT (사물 인터넷)

키워드 :
	시각장애인, 이어폰, 내비게이션, GPS

기능 :
	[1] 집 안에서 가구나 특정 물건을 코드 스캔 또는 주변
	기기 감지를 통해 등록하여, 위치가 바뀌거나 한다면
	이어폰으로 알려주는 기능, (또는 스캔 가능한 물건이
	같은 iOT 물건이라면 스피커를 통해 소리가 날 수 있으면
	좋을 것 같다.

	[2] 흰 지팡이와 연동하여 사용할 수 있으며, GPS 기능을
	통해 가는 길을 이어폰으로 알려주는 기능.
	음성 인식을 통해 목적지 설정이 가능하며, 근처에
	신호등이 있다면 음성 신호기 사용을 권장하거나,
	음성 신호기 역할을 해준다.
	본인 위치를 상대방에게 GPS 좌표를 공유해준다.

메인 페이지 : GPS 지도 + 음성 인식 버튼
서브 페이지 : 물건 등록 + 등록된 물건 확인

		 좋긴 할 것 같다.
